# Shooting Game CMS

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure environment variables:
   - Copy .env.example to .env
   - Update MongoDB connection string

3. Start the server:
   ```bash
   npm run dev
   ```

## Project Structure
```
shooting-game-cms/
```
